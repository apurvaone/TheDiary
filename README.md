# TheDiary 
A Simple offline Daily Diary application with simple minimalistic and pleasent UI.


### Tech Used ###
  * Architechture- MVVM
  * Language- Kotlin
  * UI-Material Design Library(Material.io)
  * Database- Room,firebase
